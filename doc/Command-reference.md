# Python ROM Manager command reference

## Invocation

```
prm COMMAND [SETNAME] [OPTIONS]
```

## Configuration file

The configuration file must be named `configuration.xml`. Next is an example of
`configuration.xml`:

```
<!-- Example configuration file for prm -->
<general>
    <DATdirectory></DATdirectory>
</general>

<set>
    <name></name>
    <DAT></DAT>
    <platform></platform>
    <ROMdir></ROMdir>
</set>


```
